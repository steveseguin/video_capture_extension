Method 2: Manual Installation Alternative Method

Install the extension manually by loading it as an unpacked extension in Chrome.

Download the Extension

Download the latest release ZIP file from GitHub.

Download Extension ZIP

Extract the ZIP File

Right-click the downloaded ZIP file and select "Extract All" (Windows) or double-click it (Mac) to extract the contents to a folder.

Remember where you extracted it - you'll need to select this folder in Chrome.

Open Chrome Extensions Page

Open Google Chrome and navigate to chrome://extensions/

Or go to Menu (‚ãÆ) ‚Üí More Tools ‚Üí Extensions

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the extensions page.

Load the Extension

Click the "Load unpacked" button that appears after enabling Developer mode.

Navigate to and select the folder where you extracted the extension files.

Verify Installation

The VDO.Ninja Video Capture extension should now appear in your extensions list.

You'll see the extension icon in your Chrome toolbar (you may need to click the puzzle piece icon to pin it).

Using the Extension

Navigate to a Video Page

Go to any webpage containing videos (YouTube, Twitch, Discord, etc.)

Click the Extension Icon

Click the VDO.Ninja Video Capture icon in your toolbar to open the popup.

Select a Video or Tab

Choose from detected videos or use the "Tab Capture" option to stream your entire tab.

Start Streaming

Click "Publish Video" to start streaming. You'll receive links that viewers can use to watch your stream.

Troubleshooting

Extension not working?
  • Make sure you've extracted the ZIP file completely before loading
  • Refresh the page after installing the extension
  • Check that Developer mode is enabled
  • Try reloading the extension from the Extensions page
  • If videos appear black, check if the site uses DRM protection (see limitations above)
